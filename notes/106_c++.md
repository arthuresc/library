# C++ (e as vezes uns C)

## Sintaxe

### Arquivos .h (ou .hpp)

- São arquivos de cabeçalho que servem como arquivos para serem importados.

### `#pragma once` e `#ifndef`

- Ambos garantem a unica inclusão do arquivo em uma unica compilação.
- Apesar de ambos terem a mesma função mas sintaxes diferentes:


- O `#pragma once` é mais recente e simplificado, a maioria dos compiladores já entendem

```cpp
  #pragma once

    class Animal {
        // ...
    };
```

- O `#ifndef` consegue ser usado com compiladores mais antigos mas precisa ser finalizada

```cpp
  #ifndef ANIMAL_HPP
  #define ANIMAL_HPP

    class Animal {
    // ...
    };

  #endif // ANIMAL_HPP
  ```

## Conhecimentos

### Guardar valores: #define | const | e variaveis não inicializadas

- O `#define` vai guardar um valor, que é no final tratado como **string**,

### Como importar funções, classes, estrturas e constantes

- Crie o arquivo .h
- Use o `#pragma once` ou `#ifndef`
- Crie as funções ou o que for necessário para ser exportado
