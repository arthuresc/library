# Tilix

## Comandos e atalhos

### Trocar entre os sub-terminais

`alt` + `setas`

### Redimencionar os sub-terminais

`alt` + `shift` + `setas`

### Repartir Horizontalmente

`ctrl` + `alt` + `D`

### Repartir Verticalmente

`ctrl` + `alt` + `Y`

### Fechar cada repartição

`ctrl` + `shift` + `W`

### Abrir os caminhos favoritos

`ctrl` + `shift` + `B`

### Exibir somente o sub-terminal em foco

`ctrl` + `shift` + `X`
